# Movie Explorer

A React + TypeScript application using TMDB API.

## Features
- Top Rated Movies (pagination)
- Movie Search (with year filter)
- Watchlist management
- Rating system (1–10)
- Summary analytics (charts)
- Movie Details page

## Tech Stack
- React + TypeScript
- MUI
- React Query
- Recharts
- TMDB API

## Notes
- Uses optimistic updates for watchlist
- Dark IMDb-inspired UI
